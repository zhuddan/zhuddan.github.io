断言

  1. 类型断言 (as)： 这是最常见的类型断言形式。它告诉编译器，你确切地知道一个值的类型，并希望在这个上下文中将其视为这个类型。
  1. 非空断言 (!)： 非空断言告诉 TypeScript 一个表达式的值不为 null 或 undefined。它可以用于从一个可能为 null 或 undefined 的值中去除这些类型。
  1. 类型断言函数： TypeScript 也允许通过定义特定形式的函数来执行自定义的类型断言。这种方法对于复杂的类型转换或校验非常有用。
