# docusaurus 页面配置说明

```md
---
# 基本元数据
title: 文档标题
description: 文档描述（SEO使用）
tags:
  - React
  - 前端
sidebar_label: 侧边栏显示标签
sidebar_position: 1  # 在侧边栏的排序位置

# 页面控制
hide_title: false  # 是否隐藏标题
hide_table_of_contents: false  # 是否隐藏目录

# 编辑相关
last_update:
  date: 2024-01-01  # 最后更新日期
  author: 作者名称

# 作者信息（博客专用）
authors:
  - name: 作者名
    title: 作者头衔
    url: 作者链接
    image_url: 作者头像链接

# 封面图片
image: /path/to/cover.jpg  # 分享时使用的封面图

# 草稿状态（博客）
draft: false  # 是否为草稿

# 其他
slug: 自定义-url-路径
id: 唯一标识符
---
```
