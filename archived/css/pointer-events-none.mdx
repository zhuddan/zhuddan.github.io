# 阻止所有事件

常用于水印设置/或者弹窗底部的遮罩层是否可以进行点击

```css
* {
  pointer-events: none;
}
```

## 水印示例

```css title="styles.css"
.watermark {
  ...
  /*  highlight-next-line */
  pointer-events: none;
  /*  highlight-next-line */
  background-image: url(你的水印图片)
}
```
